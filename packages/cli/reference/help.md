`rockset help`
==============

Display help for rockset.

* [`rockset help [COMMAND]`](#rockset-help-command)

## `rockset help [COMMAND]`

Display help for rockset.

```
USAGE
  $ rockset help [COMMAND] [-n]

ARGUMENTS
  COMMAND  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for rockset.
```
